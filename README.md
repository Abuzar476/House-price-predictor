# House-price-predictor
This program takes a csv file (like housing.csv) then it tells you how accurate it can predict prices.
Then it asls you for some details, for example:
Enter housholds: 5, 
Enter median income: 5, 
Enter total rooms: 20, 
Enter total bedrooms: 4, 
Enter house median age: 10, 
OUTPUT: 
Accuracy: 0.6210907494093486
203627.515 (predicted price)
Write your asking price: 200000
This is a good deal!
